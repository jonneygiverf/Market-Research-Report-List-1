<p><p>Latch ICs are integrated circuits used for storing binary data. The SR (Set-Reset) type latch has two inputs, SET and RESET, which set or reset the output respectively. The D (Data) type latch has a single input, which transfers the data at the input to the output upon a clock signal. The T (toggle) type latch, also known as a flip-flop, toggles its output state when receiving a clock signal. The J-K type latch has two inputs, J and K, which control the output based on their combination of logic levels.</p></p>
